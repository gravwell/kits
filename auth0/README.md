A toolkit for interacting with your Auth0 data. 

An [integration](https://docs.gravwell.io/integrations/integrations.html) guide for [Auth0](https://docs.gravwell.io/integrations/kitCategory/auth0.html) is available on [Gravwell Docs](https://docs.gravwell.io/).

It provides the following utilities:  
    - Actionables  
    - Dashboards  
    - Extractors  
    - Files  
    - Macros  
    - Playbooks  
    - Queries  
    - Resources  
    - Scheduled Searches  
    - Templates  

The [Auth0 Kit](https://github.com/gravwell/kits/tree/main/auth0) is licensed under the BSD 2-Clause license and the contents are available on [Github](https://github.com/gravwell/kits/tree/main/auth0).

## Dependencies  
- None
